# Customer Behavior Analysis - Alfido Tech

## Project Overview
This project analyzes customer behavior using RFM Segmentation, Purchase Pattern Analysis, and Churn Risk Assessment techniques.

## Objectives
- Customer Segmentation
- Purchase Pattern Analysis
- Retention Analysis
- Churn Risk Assessment
- Business Recommendations

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Dataset
Customer Behavior Analysis Dataset (Kaggle)

## Key Findings
- Top customers generate most of the revenue.
- Electronics category has the highest Average Order Value.
- Around 25% of customers are at churn risk.

## Recommendations
1. Loyalty Rewards Program
2. Win-back Campaigns
3. Recommendation Engine
4. Churn Prediction Dashboard
5. Cross-sell & Upsell Strategy
